The Solution

Rest + RestScan = The Closed Loop

Two products that only work together — creating a moat no competitor can replicate without building both.

Rest Device — Treat

Dual-mode periocular thermal therapy (warm + cool + pulsed). Health Canada approved. IEC 60335-1 certified. 10-minute daily sessions at home.

84% daily usage rate — 9–14× higher than any existing DED modality.

~3,000 units shipped 0.67% return rate

RestScan — Measure

Browser-based webcam biometric analysis. 120-second scan measures blink dynamics, fatigue, and autonomic markers via MediaPipe. 100% local processing — no data leaves the device.

Scan → Score → Treat → Rescan → Delta

POC stage ~12 months to clinic-ready

The moat: No CV company can replicate without building a treatment device. No DED device company can replicate without building CV infrastructure. RestScan + Rest = the only closed-loop system in ocular care.
