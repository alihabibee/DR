Section 1 · Technical Overview
The Structural Moat
Umay is the only company that combines webcam-based ocular biometric assessment with a patented thermal therapy device — creating a closed-loop system where measurement informs treatment and treatment generates measurement data. No competitor can replicate this without building or acquiring both halves.

RestScan: What It Is

RestScan is a browser-first computer vision tool that uses a standard webcam to assess ocular fatigue and autonomic stress. It runs a 120-second segmented protocol — 50 seconds for dry eye assessment, 70 seconds for fatigue gating — with all processing performed locally in the browser. No video data leaves the device.

Built on MediaPipe face mesh (open-source), RestScan adds Umay's proprietary signal processing, quality-gating architecture, and multi-domain fusion scoring (FOARA) on top.

FOARA Signal Domains

Facial-Ocular Autonomic Response Assessment (FOARA) extracts signals across three measurement domains, each with independent quality confidence thresholds.

Ocular Surface
Blink rate & completeness — DED marker
Inter-blink interval — tear film stability proxy
Partial blink ratio — meibomian dysfunction signal
PERCLOS — eyelid closure percentage (fatigue)
FDA: Class I exempt (wellness) or 510(k) QME pathway
Autonomic / Cardiovascular
Heart rate via rPPG — remote photoplethysmography
HRV estimation — autonomic balance indicator
Respiratory rate — stress/relaxation marker
Pupil dynamics — sympathetic/parasympathetic proxy
FDA: 510(k) QME (HR/RR), Class I exempt (pupillometry)
Behavioral / Temporal
Gaze stability — attention / fatigue marker
Fixation drift — oculomotor fatigue
Micro-expression patterns — stress behavioral signal
Session-over-session delta — longitudinal trend
No FDA pathway required (behavioral analytics)

The Replication Problem
To replicate Umay's position, a CV company (Nuralogix, Smart Eye) must build or acquire a thermal therapy device, navigate Health Canada/FDA clearance, and build a clinic channel from zero. A DED device company (Johnson & Johnson Vision, Alcon) must build computer vision capability, hire ML talent, and obtain biometric data consents at scale. Neither path is fast or cheap.

Data Flywheel Logic

Each device sold generates usage data. Each RestScan session generates biometric data. Combined, they create a paired dataset (thermal therapy input → ocular/sleep output) that no competitor can replicate without:

1. Selling a competing thermal device
2. Through a comparable clinic channel
3. With integrated biometric measurement
4. Over comparable timeframes

This dataset is the foundation for future ML-driven personalization: "Your blink metrics suggest 8 minutes of pulsed mode at 42°C tonight."

Dual-Use IP Leverage: OARA Defence Variant

The same core technology that powers RestScan has defence applications. Umay is exploring an NRC IRAP-funded project (OARA — Ocular Autonomic Response Assessment) for military fatigue monitoring. The R&D investment serves both markets: commercial RestScan for clinics and consumers, OARA for defence and industrial safety. One engineering effort, two revenue channels — with government co-funding reducing the cost base for both.
